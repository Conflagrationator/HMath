Description

Predefined SNat singleton literals in the range [0 .. 1024]

Defines:

d0 = SNat :: SNat 0
d1 = SNat :: SNat 1

...

You can generate more SNat literals using decLiteralsD from CLaSH.Promoted.Nat.TH
